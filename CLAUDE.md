# AIDigestCN — Copilot 指南

## 项目概述

每日自动抓取顶级 AI 领袖推文，翻译成中文，发布到 GitHub Pages 的资讯站点。

## 设计系统

在做任何视觉或 UI 决策之前，请先阅读 `DESIGN.md`。

所有字体选择、配色、间距、圆角和美学方向均在 `DESIGN.md` 中定义。未经用户明确批准，不得偏离这些规范。

在 QA 模式下，请标记任何不符合 `DESIGN.md` 的代码。

**核心原则：**
- 以人物为主角（头像/姓名是视觉焦点）
- 强调色使用森林绿 `#15803D`，不是蓝色
- 刊头使用 Clash Grotesk 几何粗体
- 正文使用 Source Sans 3，中文回退至 PingFang SC 系统字体栈
- 深色模式背景使用微绿近黑 `#0A0F0D`，不是普通深灰
